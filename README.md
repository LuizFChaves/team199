# FrontEnd da plataforma Rango

BackEnd feito em React para o MVP da aplicação Rango, participante no grupo 199 do Hackathon CCR 2020.

## Versionamento

Por favor, consulte nosso [CONTRIBUTING](CONTRIBUTING.md).

## API

Install

```
npm install
```

Run with Node

```
npm start
```

Run with Nodemon

```
npm run dev
```

Run ESlint

```
npm run lint
```
